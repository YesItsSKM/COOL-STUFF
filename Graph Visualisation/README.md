# Graph Visualisation

C++ program using GraphViz to visualize weighted or non-weighted directional or undirectional graphs.

Input(Need to enter in program):
  1. Number_of_nodes.
  2. Starting & ending node of edge(s).

Output:
  .png file of the graph.
